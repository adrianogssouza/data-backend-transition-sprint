# Data Backend Transition Sprint

Sprint intensiva focada em **Python, SQL, FastAPI, PostgreSQL e dbt**, com o objetivo de consolidar uma base prática em **backend orientado a dados** e **analytics engineering**.

## Objetivo

Este repositório documenta minha jornada prática de transição para as áreas de **Engenharia de Dados** e **Desenvolvimento Backend**, por meio de uma sprint estruturada de estudos, prática e construção de projetos.

A proposta é sair do estudo passivo e construir uma base técnica aplicada em:

- Python para dados e backend
- SQL para análise e performance
- FastAPI para desenvolvimento de APIs
- PostgreSQL para modelagem relacional
- dbt para analytics engineering
- Docker como parte do fluxo de entrega da aplicação
